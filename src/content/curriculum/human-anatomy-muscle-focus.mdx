## Muscle system overview

Skeletal muscles create movement, stabilize joints, and support posture. Learning key groups by movement pattern is more useful than memorizing names in isolation.

## Major muscle groups

| Region | Major muscles | Primary actions |
| --- | --- | --- |
| Chest | Pectoralis major/minor | Horizontal push, shoulder movement |
| Back | Latissimus dorsi, trapezius, rhomboids | Pulling, scapular control |
| Shoulders | Deltoids, rotator cuff | Shoulder rotation and arm elevation |
| Core | Rectus abdominis, obliques, transverse abdominis | Trunk stability and rotation |
| Legs | Quadriceps, hamstrings, gluteals, calves | Squat, hinge, gait, propulsion |

## Learning strategy

1. Pair each muscle group with one movement pattern.
2. Study origin/insertion only after basic function is clear.
3. Use simple diagrams and teach-back explanations.
4. Review with short daily sessions instead of long cramming.

<Alert tone="info" title="Practical study tip">
Use movement-based notes like: "hamstrings = hip extension + knee flexion" to improve recall in clinical or coaching settings.
</Alert>

## Functional movement examples

- **Squat pattern**: quadriceps and gluteals dominate.
- **Hinge pattern**: hamstrings and gluteals lead with spinal stability.
- **Push pattern**: chest, shoulders, triceps.
- **Pull pattern**: back, rear deltoids, biceps.
