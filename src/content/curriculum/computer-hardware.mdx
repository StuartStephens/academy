## Why study computer hardware?

Computer hardware determines how systems process, store, move, and display information. Understanding physical components makes it easier to troubleshoot systems, plan upgrades, and understand software performance limits.

## Core components

| Component | Main role |
| --- | --- |
| CPU | General computation and instruction execution |
| RAM | Short-term working memory |
| Storage | Persistent data retention |
| Motherboard | Electrical and data interconnection platform |
| GPU | Parallel graphics and compute workloads |
| Power supply | Stable power delivery to system components |
| Cooling | Thermal control and reliability protection |

## Core questions

- What component is the performance bottleneck?
- How do bandwidth, latency, and heat affect behavior?
- Which parts are upgradeable or replaceable?
- How do workloads differ between gaming, servers, editing, and general use?

## Common study areas

1. Component roles and interoperability.
2. Desktop vs laptop vs server design constraints.
3. Basic diagnostics and troubleshooting.
4. Thermal management and power stability.

<Alert tone="info" title="Hardware principle">
Performance is not just about the fastest part. Balanced systems usually outperform mismatched builds for real-world tasks.
</Alert>
