## Why the periodic table matters

The periodic table organizes elements by atomic structure and recurring chemical behavior. It is one of the most important tools for understanding matter, reactions, bonding, and material behavior.

## Core topics

| Topic | Why it matters |
| --- | --- |
| Atomic number | Defines the identity of an element |
| Groups and periods | Show recurring patterns in chemical behavior |
| Metals, nonmetals, metalloids | Broadly organize material properties |
| Valence behavior | Helps explain bonding and reactivity |
| Periodic trends | Atomic radius, electronegativity, and ionization energy guide prediction |

## Main regions to know

1. Alkali metals.
2. Alkaline earth metals.
3. Transition metals.
4. Halogens.
5. Noble gases.
6. Lanthanides and actinides.
