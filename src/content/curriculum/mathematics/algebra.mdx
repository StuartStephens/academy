## Why algebra matters

Algebra turns arithmetic patterns into general rules. It allows relationships to be expressed symbolically and solved across many different situations.

## Core topics

| Topic | Why it matters |
| --- | --- |
| Variables and expressions | Represent changing or unknown quantities |
| Equations | Formalize relationships and solve for unknowns |
| Functions | Describe how one quantity depends on another |
| Graphs | Make relationships visible and interpretable |
| Manipulation rules | Correct symbolic work prevents false conclusions |
